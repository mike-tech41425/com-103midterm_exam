To be able to see and use the code make sure you copy the code inside the repository and phase it to the VS code or any type of online python to put your code.
this are the steps to follow.
**Step 1: Initial Setup**
Enter Name: Type the student's name when prompted.
Set Budget: Enter the total amount of money available for the week (as a number).
**Step 2: Category Selection**
The program will display a menu of 5 categories. For each of the 4 entries, you must enter a number:
**1 to 5** To select a specific category (Food, Transport, etc.).
**0** To skip that specific entry slot.
Any other number: Will be treated as an invalid choice and the slot will be skipped.
**Step 3 Entering Expense Details**
If you chose a valid category (1–5):
Description: Type a short note about what you bought (e.g., "Chicken Joy" or "Jeepney fare").
Amount:Enter the exact amount spent on that item.
**Step 4: Reviewing the Final Report**
After 4 attempts to enter data, the program prints a Final Expense Log containing:
The Logged List: A list showing the entry number, category, description, and price.
High Expense Alert: Any item costing more than 25% of your total budget will be marked with ! High Expense Alert!.
Financial Summary: The total amount spent and the remaining balance.
**Budget Status:**
Budget OK! Keep it up. (If you have money left or exactly zero).
Overspent! Reduce spending. (If you spent more than your budget).
